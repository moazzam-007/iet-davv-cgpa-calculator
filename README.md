# IET DAVV CGPA Calculator 🎓

A modern, mobile-responsive, and offline-capable CGPA & SGPA calculator designed specifically for the students of IET DAVV. 

Built with a highly tactile **LCD Scientific Calculator** UI design. 

### 🌐 Visit the Calculator
👉 **[Click here to use the Calculator Live!](https://moazzam-007.github.io/iet-davv-cgpa-calculator/)**

## Features
- **Auto-Save (LocalStorage):** Never lose your data; it saves locally on your device as you type.
- **SGPA to CGPA Sync:** Easily copy calculated term SGPA directly into the CGPA calculator.
- **Mobile Optimized:** Full-width native app-like experience with large touch targets.
- **Customizable:** By default, it loads the E&TC syllabus, but any student from any branch can edit the subject names and credits for their use!
